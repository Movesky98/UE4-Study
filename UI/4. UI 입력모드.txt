# 29. UI 입력모드 변경하기

# 강의 목표
- 메뉴를 클릭할 수 있도록 만들것임
- 게임 내에서 마우스 커서가 보이도록 하여, 클릭할 수 있도록 만들것임.

# 플레이어 컨트롤러에 변화를 줘야 한다.
- 왜 플레이어 컨트롤러인가?
- 플레이어 컨트롤러가 현재 게임으로 이동해온 것이기 때문에
- APlayerController::SetInputMode를 이용하여 입력모드를 변화시킬 수 있다.
- MouseCursor를 보이도록 만들고, FInputModeDataBase를 사용하여 몇몇 설정을 해주어
- UI를 입력할 수 있는 상태로 만들어 주어야 한다.
