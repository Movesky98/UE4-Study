25. 메뉴 블루프린트 작성

# 게임에 메뉴 시스템을 구축
- 멀티플레이를 위한 간단한 메뉴 레이아웃 작성

# 메뉴 시스템은 어떻게 작동하는가!
- 메뉴 시스템은 게임으로부터 완전히 분리 가능하고, 쉽게 분리될 수 있다.
- 이 강의에서는 게임과 분리된 메뉴 시스템을 구축하려 함.


# 위젯 블루프린트 생성
- [콘텐츠 브라우저] -> [추가/임포트] -> [User Interface] -> [Widget Blueprint]
- 위젯 블루프린트는 접두사로 WBP_를 붙여 기존의 블루프린트와 차별점을 두어야 한다.

# 앵커
- Canvas Panel (레이아웃) 상에서, 어디에 중심을 둘 지 결정할 수 있다.
- 앵커는 화면의 크기(레이아웃의 크기)가 달라졌을 때, Button이나 Text 같은 컴포넌트들을
  같은 위치상에 나타내기 위해(비율적으로) 사용됨.
