31. 커스텀 버튼 및 폰트

# 커스텀 글꼴, 버튼을 정의하고 배경을 추가하여 메인메뉴를 확장시켜보자.
- 어떻게 하면 멋있게 보일 수 있을까
- 배경, 버튼 / 폰트의 디자인

# 전체적으로 외부에서 가져온 이미지를 활용해 배경, 버튼, 폰트를 디자인하는 작업을 하였음.
- 버튼의 스타일에 대해서도 설명이 나온다.
- Box / Border / Image에 따라 어떤 형태로 그릴 것인지 지정할 수  있다고 함.
- 자세한 설명은 아래 참고자료에 적혀있음

# 다양한 폰트를 찾아보기 위해 Google Fonts를 이용할 수 있다.
- Serif : 글자 끝에 뾰족한 부분이 있다는 것을 의미 (New Roman에서 따옴)
- Sans Serif : 뾰족한 부분이 없는 글꼴
- Handwriting : 손글씨
- Monospace : 고정너비 글꼴, 터미널 및 프로그래밍 편집기에서 사용됨
- Display : 간지용

# 위젯 블루프린트를 테스트할 때, 창 크기를 조절하다보면 기이하게 변하는 것이 보임 
- 이를 방지하기 위해 Scale Box를 사용함

# Font 바꾸기
- 원하는 폰트의 ttf 파일을 받은 후 별다른 작업 없이
- 새로운 폰트 에셋을 만들 것이냐는 메세지 창에서 "예" 선택
- 이후 [위젯 블루프린트] -> [원하는 위젯] -> [Font Family 설정]
