33. 언리얼 모션 그래픽 (UMG)를 C++에 연결하기

# 위젯 블루프린트의 일부를 C++로 다시 지정하여 기능을 추가해보자.
- C++ 프로퍼티를 UI 버튼에 바인딩할 것임
- 실제로 버튼을 눌러 이벤트가 발생하게 하는 것은 C++을 통해 만들 수 있다.

# 위젯에 C++ 부모 클래스 만들기
- C++을 통해 위젯 블루프린트의 로직을 관리할 수 있다.
- 위젯 블루프린트의 부모 클래스를 C++로 바꾼 후
- 프로퍼티를 Button에 바인딩
- C++에 UPROPERTY로 버튼을 생성하면 우리가 만든 위젯에 자동으로 바인딩 됨.
- 블루프린트에 배치한 버튼과 상호작용하는 것을 쉽게 할 수 있으며
  바인딩 코드가 거의 필요하지 않게 됨.

# 위젯 블루프린트의 부모 클래스 설정
- [위젯 블루프린트 실행] -> [우측상단] -> [Designer > Graph] 중 Graph 선택
  [Class Settings] -> [Details] -> [Parent Class]

# BindWidget 만드는 법
@ Bind Widget이란?
 - UPROPERTY(BlueprintReadWrite, meta = (BindWidget))
   UTextBlock* ItemTitle; (또는 버튼)
 - 이런식으로 선언한다면 서브 위젯에 있는 위젯에 자동으로 어태치(Attach)해준다.
 - 또, 위젯 블루프린트에서 이미 버튼 / 텍스트의 배치를 한 후
   부모 클래스로 바꾸어 빌드를 했다면, 해당 위젯 블루프린트에서 컴파일이 안되는 것을       볼 수 있다.
 - 이는 지정한 프로퍼티 이름과 동일한 이름으로 위젯을 구현해야함을 보여준다.
