# 다수의 플레이어로 테스트하는 방법

[언리얼 엔진 클라이언트] -> [플레이 드롭다운] -> [플레이어 수 변경]
[언리얼 엔진 클라이언트] -> [플레이 드롭다운] -> [넷모드 - Play As Listen Server]
-> 이렇게 하고 [플레이]를 누르면 리슨서버에서 2명의 플레이어가 생성된 것을 볼 수 있다.

프로젝트에서 [플레이] 버튼을 눌렀을 때 실행되는 과정을 들여다보면
1. 언리얼은 내부적으로 '맵'을 로딩하게 된다.
2. 에디터에 맵이 로드되고 [플레이] 버튼을 클릭하면 실행할 수 있게됨
3. '맵'은 [게임 모드]를 특정할 수 있음
 - [블루프린트] -> [Project Settings - 게임모드] or [World Override - 게임모드]
 - 월드랑 맵은 다른 것인가 ?
   -> 거의 비슷하다. 다만 월드에 좀 더 많은 정보가 들어가있음.

4. '맵'에 참여하려는 [플레이어의 컨트롤러]를 얻음
5. [플레이어 컨트롤러]가 [게임모드]에 요청하여 [Pawn]을 생성하도록 함
 - Pawn 이란 ?
   -> 플레이어나 AI가 제어할 수 있는 모든 액터의 베이스 클래스

6. [Pawn]이 생성되면, [플레이어컨트롤러]와 연결되어 [Pawn]을 직접 조종할 수 있게 됨
