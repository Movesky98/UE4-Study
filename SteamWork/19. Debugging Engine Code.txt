73. 엔진 코드 디버깅하기.

# 목표
- 엔진 코드 디버깅을 통해, null Subsystem에서도 접속중인 플레이어 수를 확인해보자.
- nullSubsystem 에서 버그를 찾아 고쳐버리자잇

# 디버깅을 위한 다운로드
- [에픽 게임즈 런처] -> [Unreal Engine 4] -> [Library] -> [자기 버전 실행 옆 버튼] 
  -> [Option] -> [디버깅을 위한 편집기 기호 체크]

# NullSubsystem에서 발생한 오류이므로 여기에서 실행된 것으로 보이는 함수를 이용
- Ctrl + Shift + F 를 이용해 프로젝트 전체 검색을 수행

# 중단점을 이용한 디버깅
- 함수의 위치를 찾고 중단점을 설정했다면, 
  Ctrl + Shift + P 를 이용해서 실행중인 게임 인스턴스에 연결함. (게임이 실행중이어야 함.)
- 이후 해당 함수가 실행되거나 중단점 지점을 지날 때, 멈추게되어 확인할 수 있음.
