60. 서버 리스트 채우기

# 목표
- 실제로 호스팅되고 있는 세션을 서버 리스트에 보이도록 해보자
- 이를 이용하면 나중에 졸업작품 게임프로젝트에 매치메이킹에 도움이 많이 될듯?
- 어드민 관점에서 바라본 UI 또한 설계했어야 했는데 이걸로 쉽게 도움받을 수 있음.

# 해야할 일
1. 서버리스트에 들어올 행(Row) 아이템에 텍스트 프로퍼티를 설정할 수 있는지 확인
2. GameInstance에서 서버리스트에 서버 문자열 리스트를 추가하기 위한 메서드 만들기
3. GameInstance에서 Refresh(새로고침) 요청할 수 있어야 함.

# 띵팁
- 위젯 컴포넌트의 C++ 자료형을 알기위한 꿀팁
  해당 위젯 컴포넌트를 클릭 -> 디테일 창 우측 상단 @@ 열기 클릭하면 알 수 있음.

# UTextBox::SetText(FText)
- FText::FromString(FString)을 이용하여 텍스트 박스의 내용을 설정 가능
